class *Resource*(object)<br>
Base class for remote objects that the Scribd API allows<br>
to interact with.<br>
<br>
This class is never instantiated, only subclassed.<br>
<br>
Every object features a set of resource attributes that are<br>
stored and managed by the Scribd platform. They are accessed<br>
and used like any other Python object attributes but are<br>
stored in a separate container.<br>
<br>
Methods defined here:<br>
<br>
*get_attributes*(self)<br>
Returns a dictionary with the resource attributes. </pre>
